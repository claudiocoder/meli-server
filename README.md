# MELI NODE SERVER CMORENO

## How to install Server

### Install

```bash
npm install
```

### Run Server

```bash
npm run dev
```

- Default port
  localhost:3000

### Run Test

```bash
npm run test
```

### RUN Coverage

```bash
npm run cover
```

### OPEN Report

```bash
npm run report
```

### Include

- Postman Collection
- Server Production: Heroku
